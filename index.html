User interface design
for streaming application

In this project, we had to work in a group and design an application for a hypothetical brand. These days, there are a lot of streaming applications available but they don't have features like user reviews, creating lists, advanced search and filter options. Users have often ask friends/family or visit third-party websites to search for their preferred movies and series to watch. This was a problem statement, around which we developed the concept of our brand "Ubinge".

User interface design
for website

For this project, I designed the user interface for the website for my hypothetical brand "Little Edisons". Little Edisons is a brand which conducts interactive science workshops, live stage shows that brings excitement for science amongst children, at different locations throughout the country. A website was needed so that the target users could easily search through the available workshops and book their slot according to their location and topic of interest. The process involved creating user personas, sitemaps, wireframes, final screen designs and prototyping.

Brand identity design
for sports sector

For this project, I created a hypothetical brand in the sports sector. My brand "Get Set Play" is a brand for sports players which sells used/unused branded second-hand sports equipment for lower prices (e.g. tennis rackets, running shoes, swimming goggles, badminton rackets). Anyone will be able donate their used/unused branded sports equipment, which will be targeted towards sportspersons, schools and sports academies. The project involved creating a whole brand identity, collaterals, brand style guide, adverting campaign, promotional media, catalogue, merchandise, etc. for the brand.
